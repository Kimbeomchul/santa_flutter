# santa_flutter


### how to use
```
sudo pub get
sudo flutter run --no-sound-null-safety
```

## Santa 구성

- Native : Flutter ( IOS , Android 위치정보 스프링으로 리턴 ) 

- F :  Thymeleaf , Vue js 

- B : SpringBoot 

- CI/CD : Docker , Jenkins , AWS (EC2 , RDS ,S3)

- DB : MySQL

- Design : Figma 

- 주피터노트북 시스템 데몬설정 port 8888
- 스프링부트 로그인필터설정
- 스프링부트 세션리스너설정
